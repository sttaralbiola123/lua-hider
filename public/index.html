How It Works

Unique Script IDs

When you upload a Lua script, Lua Hider generates a unique 8-character alphanumeric ID. This ID is tied to your script and stored securely on the backend server.

Browser Protection

When someone opens the script URL in a regular web browser, they are redirected to a Rickroll page. This protects the script contents from being viewed by unauthorized users browsing to the URL manually.

Roblox Executor Access

Roblox script executors use a raw HTTP request to fetch the Lua script directly. The backend detects executor requests by checking headers and serves the raw Lua code, bypassing the Rickroll protection. This allows legitimate use inside Roblox.

Loadstring Format

The generated loadstring follows the standard Roblox executor format:

loadstring(game:HttpGet("https://lua-hider.onrender.com/raw/lua/:id/download"))()
Usage Guide

Step-by-Step Instructions

  1. Paste your Lua script into the Script Editor text area.
  2. Click "Hide Script" to upload it to the Lua Hider server.
  3. Copy the generated loadstring from the result block that appears.
  4. Open your Roblox executor (e.g. Synapse X, KRNL, Fluxus, etc.).
  5. Paste the loadstring into the executor's script input box.
  6. Click Execute in your executor to run the hidden script.

Tips

Your recent scripts are saved in the "Recent Scripts" section for quick access. You can copy any previous loadstring from there without re-uploading.

Use the AI Assistant to analyze, debug, or review your scripts before hiding them for best results.
