About LuaBin
Store and host your Roblox Lua scripts in the cloud – no file sharing needed.
Each script gets a unique 8‑character link that only works from Roblox executors.
Browser access is blocked (404) to keep your scripts private.
Scripts are stored securely in Supabase and served with no‑cache headers.
